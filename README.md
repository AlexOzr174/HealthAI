# HealthAI - Умный гид по здоровому питанию

**HealthAI** — это десктопное приложение на Python для отслеживания питания, планирования рациона и получения персональных рекомендаций по здоровому питанию. Приложение сочетает медицинские стандарты (диеты по Певзнеру) с современными технологиями персонализации.

## ✨ Возможности

### 🤖 AI-персонализация
- **Умный онбординг**: Первичная анкета для расчёта индивидуальной нормы калорий
- **Расчёт по формуле Миффлина-Сан Жеора**: Точный расчёт базального метаболизма (BMR) и суточной нормы (TDEE)
- **Адаптивные рекомендации**: Система подстраивается под ваши предпочтения

### 📔 Дневник питания
- Быстрое добавление продуктов с автоподсчётом калорий
- Отслеживание БЖУ (белки, жиры, углеводы)
- История приёмов пищи за любой день
- Визуализация прогресса в реальном времени

### 🏥 Медицинские диеты по Певзнеру
- **15 лечебных столов**: От №1 до №15
- Автоматическая фильтрация рецептов по диете
- Предупреждения о запрещённых продуктах
- Подробные рекомендации для каждой диеты

### 📅 Планировщик питания
- План питания на неделю
- Функция автозаполнения на основе рекомендаций
- **Автоматическая генерация списка покупок**
- Экспорт списка в текстовый файл

### 📚 База рецептов
- 50+ готовых рецептов с расчётом на порцию
- Фильтрация по категориям (завтрак, обед, ужин, перекус)
- Поиск по названию и ингредиентам
- Информация о калорийности и БЖУ

### 🎮 Геймификация
- Система уровней и опыта (XP)
- **10+ достижений** для разблокировки
- Отслеживание серии дней (streak)
- Водный баланс с напоминаниями

### 💡 Рекомендательная система
- Персонализированные рекомендации рецептов
- Анализ nutritional gaps (недостающих нутриентов)
- Здоровые замены для вредных продуктов

## 🚀 Установка

### Требования
- Python 3.9 или выше
- pip (менеджер пакетов Python)
- Минимум 100 МБ свободного места

### Шаг 1: Клонирование или скачивание

Скопируйте файлы проекта в директорию по вашему выбору:

```bash
cd /path/to/your/project
```

### Шаг 2: Создание виртуального окружения (рекомендуется)

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python3 -m venv venv
source venv/bin/activate
```

### Шаг 3: Установка зависимостей

```bash
pip install -r requirements.txt
```

Или установите пакеты вручную:

```bash
pip install PyQt6==6.6.1
pip install SQLAlchemy==2.0.25
pip install pandas==2.1.4
pip install numpy==1.26.3
pip install matplotlib==3.8.3
pip install scikit-learn==1.4.1
pip install PyYAML==6.0.1
```

## ▶️ Запуск

### Windows
```bash
python healthai/main.py
```

### macOS / Linux
```bash
python3 healthai/main.py
```

Или из директории проекта:
```bash
cd healthai
python main.py
```

## 🏗 Архитектура

### Модульность
Приложение построено по модульному принципу с чётким разделением ответственности:

- **Config**: Централизованные настройки и константы
- **Database**: Работа с данными и их хранение
- **Core**: Бизнес-логика и алгоритмы
- **UI**: Пользовательский интерфейс

### База данных
Используется SQLite для локального хранения:
- Пользователи и их профили
- Продукты и их питательная ценность
- Рецепты с расчётом на порцию
- Приёмы пищи (дневник)
- Достижения и их разблокировка
- Планы питания на неделю

### Расчёт калорий
```
BMR (мужчины) = 10 × вес + 6.25 × рост − 5 × возраст + 5
BMR (женщины) = 10 × вес + 6.25 × рост − 5 × возраст − 161

TDEE = BMR × коэффициент_активности

Целевые калории = TDEE × (1 + модификатор_цели)
```

### Рекомендательная система
Content-based filtering на основе:
- Истории приёмов пищи
- Предпочтений по категориям
- Типа диеты пользователя
- Рейтинга рецептов
