# 🎨 Новые UI компоненты HealthAI

## Обзор

В этом документе описаны новые страницы настроек, добавленные в приложение HealthAI.

## 📁 Созданные файлы

### 1. `settings_notifications.py` — Настройки умных уведомлений
**Строк кода:** 228  
**Классы:**
- `NotificationSettingItem` — Один элемент настройки (чекбокс + время + описание)
- `NotificationsSettingsPage` — Полная страница настроек уведомлений

**Функционал:**
- Настройка времени напоминаний о приемах пищи (завтрак, обед, ужин, перекус)
- Настройка напоминаний о воде, взвешивании, отбое
- Мотивационные уведомления (дневная цитата, прогресс цели)
- Группировка настроек по категориям
- Визуальный эффект сохранения

**Группы настроек:**
1. **Приемы пищи** (4 уведомления)
2. **Вода и здоровье** (3 уведомления)
3. **Мотивация** (2 уведомления)

---

### 2. `settings_diets.py` — Специальные диеты
**Строк кода:** 315  
**Классы:**
- `MacroCircleWidget` — Круговой прогресс-бар для макросов (custom paint)
- `DietModeCard` — Карточка режима диеты с переключателем
- `SpecialDietsPage` — Страница управления диетами

**Функционал:**
- Переключение режимов питания (Кето, Палео, IF, Веганство, Вегетарианство)
- Настройка интервального голодания (слайдер 6-12 часов)
- Визуализация целевых макронутриентов (круговые диаграммы)
- Автоматическая корректировка целей БЖУ при смене диеты

**Поддерживаемые диеты:**
| Диета | Жиры | Белки | Углеводы |
|-------|------|-------|----------|
| Кето | 70% | 25% | 5% |
| Палео | 40% | 35% | 25% |
| Веганство | 30% | 20% | 50% |

---

### 3. `settings_api.py` — Интеграция с внешними API
**Строк кода:** 346  
**Классы:**
- `APIServiceCard` — Карточка сервиса API с настройками
- `APIIntegrationPage` — Страница управления интеграциями

**Функционал:**
- Подключение внешних сервисов (Edamam, USDA, Spoonacular, Nutritionix)
- Тестирование подключения к API
- Сохранение API ключей (с маскировкой ввода)
- Переключатель видимости API ключа
- Ссылки на документацию разработчика
- Глобальные настройки (авто-синхронизация, кэширование, приоритеты)

**Поддерживаемые API:**
1. **Edamam Nutrition API** — 1M+ продуктов
2. **USDA FoodData Central** — Официальная база США
3. **Spoonacular Food API** — 500K+ рецептов
4. **Nutritionix API** — Данные ресторанов и брендов

## 🔗 Интеграция в главное окно

Для добавления новых страниц в главное окно приложения:

```python
from ui.pages import NotificationsSettingsPage, SpecialDietsPage, APIIntegrationPage

# В main_window.py добавить методы для создания страниц:
def create_notifications_page(self):
    page = NotificationsSettingsPage()
    page.settings_saved.connect(self.save_notification_settings)
    return page

def create_diets_page(self):
    page = SpecialDietsPage()
    page.diet_settings_changed.connect(self.update_diet_goals)
    return page

def create_api_page(self):
    page = APIIntegrationPage()
    return page
```

## 🎯 События (Signals)

### NotificationsSettingsPage
- `settings_saved(dict)` — Эмитится при сохранении настроек
  ```python
  {
      'breakfast': {'enabled': True, 'time': '08:00'},
      'lunch': {'enabled': True, 'time': '13:00'},
      'water': {'enabled': False, 'time': '10:00'},
      ...
  }
  ```

### SpecialDietsPage
- `diet_settings_changed(dict)` — Эмитится при изменении режима диеты
  ```python
  {
      'active_diets': ['keto', 'if'],
      'if_window': 8  # часов питания
  }
  ```

### APIIntegrationPage
- `connection_tested(str, bool)` — Результат теста подключения
- `settings_saved(str, dict)` — Сохранение настроек сервиса
