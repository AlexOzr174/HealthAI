**Файлы:**
- `ui/components/search_widget.py` - виджет поиска

## 🎯 Использование

### Переключение темы:
```python
from ui.styles import set_theme, toggle_theme, COLORS

# Установить тёмную тему
set_theme('dark')

# Переключить тему
new_theme = toggle_theme()

# Проверить текущую тему
print(f"Текущая тема: {COLORS['background']}")
```

### Умный поиск:
```python
from ui.components.search_widget import SmartSearchWidget

# Создать виджет
search = SmartSearchWidget(placeholder="Поиск рецептов...")

# Установить данные
search.set_search_data([
    {'name': 'Яблоко', 'category': 'Фрукты', 'type': 'product'},
    {'name': 'Борщ', 'category': 'Обед', 'type': 'recipe'}
])

# Обработка поиска
search.search_triggered.connect(lambda q: print(f"Поиск: {q}"))
```
