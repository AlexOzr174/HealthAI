## ✅ Рекомендации

### Для пользователей macOS
1. Установите Python через Homebrew: `brew install python@3.12`
2. Запустите: `chmod +x launch.sh && ./launch.sh`
3. При необходимости настройте URL моделей в скрипте

### Для пользователей Linux
1. Установите системные пакеты: `sudo apt install python3 python3-venv python3-pip`
2. Запустите: `chmod +x launch_linux.sh && ./launch_linux.sh`
3. Следуйте подсказкам по недостающим пакетам

### Для пользователей Windows
1. Установите Python с https://www.python.org/downloads/
2. **Важно:** Отметьте "Add Python to PATH"
3. Запустите двойным кликом `launch.bat`

### Для разработчиков
1. Настройте реальные URL моделей в массиве `MODEL_URLS`
2. Раскомментируйте команды `curl` для загрузки
3. Протестируйте на чистых системах каждой ОС
